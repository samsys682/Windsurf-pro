# Windsurf Pro

[English](README_EN.md)

Windsurf Pro 是一个面向 Windows 的桌面客户端，重点解决 Windsurf 账号切换、会话接管和本地状态管理这些高频操作，让整个流程更稳、更快、更省事。

它不是一组临时脚本的拼接，而是一个可以长期使用的桌面工具。打开后就能进入工作流，切换完成后可以继续使用，遇到异常时也能拿到更清晰的反馈。

## 当前发布

- 版本：`v0.6.2`
- 平台：`Windows`
- 发布方式：GitHub Releases / Gitee 同步仓库
- 当前产物：`Windsurf-Pro.exe`

## 适合谁用

- 需要频繁切换 Windsurf 账号
- 希望把本地状态处理收敛成固定流程
- 不想在多个脚本、目录和配置之间来回切换
- 希望桌面侧操作更可预期、更少人工确认

## 核心特点

- 更快完成账号切换，减少重复性的手动处理
- 更稳地接管本地会话和状态写入
- 常用操作集中在桌面界面里完成
- 遇到异常时更容易判断当前卡在什么阶段

## 使用提示

1. 下载最新的 `Windsurf-Pro.exe`
2. 直接运行程序
3. 按界面提示完成切换或状态处理

如果需要排查问题，可优先关注默认运行目录：

- 数据目录：`C:\ProgramData\WindsurfSwitcher`
- 日志文件：`C:\ProgramData\WindsurfSwitcher\logs\app.log`
