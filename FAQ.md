# FAQ

## 这是源码仓库吗

不是。当前公开发布面以可执行文件和配套说明为主，源码和内部开发资料不作为发布内容的一部分。

## 从哪里下载

请优先从 GitHub Releases 下载最新的 `Windsurf-Pro.exe`。如果访问 GitHub 不方便，可以使用 Gitee 同步仓库。

## 为什么现在只保留一个 exe

当前发布策略只保留最新版本，避免历史备份文件堆在公开发布目录里，减少误下旧版本的情况。

## 还会提供 zip 包吗

目前不再把 `Windsurf-Pro.zip` 作为正式发布内容保留。

## 程序是免安装的吗

当前公开产物是单文件 `exe`，下载后可直接运行。

## 遇到问题时看哪里

可以优先检查下面两个位置：

- 数据目录：`C:\ProgramData\WindsurfSwitcher`
- 日志文件：`C:\ProgramData\WindsurfSwitcher\logs\app.log`

## 为什么杀毒软件可能会更敏感

单文件桌面程序、打包器生成的可执行文件，以及涉及本地状态处理的工具，有时会触发更严格的启发式检测。遇到这类情况时，建议先核对来源和版本，再决定是否放行。
