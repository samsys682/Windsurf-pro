# Windsurf Pro

[中文](README.md)

Windsurf Pro is a Windows desktop client focused on Windsurf account switching, session takeover, and local state handling. The goal is to make repeated daily operations faster, steadier, and easier to manage.

It is built as a complete desktop workflow rather than a loose collection of scripts. Open it, perform the switch, keep working, and get clearer feedback when something needs attention.

## Current Release

- Version: `v0.6.2`
- Platform: `Windows`
- Distribution: GitHub Releases / mirrored Gitee repository
- Current artifact: `Windsurf-Pro.exe`

## Best Fit

- Users who switch Windsurf accounts frequently
- Workflows that depend on consistent local session handling
- People who want fewer moving parts and less setup friction
- Repeated desktop use cases where predictability matters

## Highlights

- Faster switching with less repetitive manual handling
- More reliable local state writing and session handoff
- A cleaner desktop flow for repeated use
- Clearer feedback when a step needs confirmation or recovery
